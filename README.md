# Precios vivienda

Proyecto de análisis y predicción de precios de vivienda basado en un conjunto de datos de inmuebles.

## Descripción

Este proyecto tiene como objetivo comprender y preparar los datos de viviendas para su uso en modelos predictivos. A partir de un dataset con características de inmuebles, se realiza un análisis exploratorio y un proceso de preprocesamiento para asegurar la calidad de los datos.

## Contexto de los datos

Los datos incluyen variables relevantes relacionadas con propiedades residenciales, como ubicación, tamaño, número de habitaciones, antigüedad y otros atributos que influyen en el valor de la vivienda.

## Trabajo en el notebook de preprocesamiento

En el notebook de preprocesamiento se llevaron a cabo las siguientes tareas:

- Limpieza de datos y tratamiento de valores faltantes.
- Detección y corrección de inconsistencias.
- Transformación de variables categóricas y numéricas.
- Escalado y normalización cuando fue necesario.
- Preparación de conjuntos de entrenamiento y prueba.

## Uso del proyecto

Este repositorio está orientado a construir un flujo de trabajo reproducible donde el notebook de preprocesamiento sirve como paso clave antes del desarrollo de modelos de machine learning o análisis estadístico.

## Estructura esperada

- `notebooks/`: contiene el notebook de preprocesamiento.
- `data/`: datos originales y/o procesados.
- `README.md`: portada explicativa del proyecto.
